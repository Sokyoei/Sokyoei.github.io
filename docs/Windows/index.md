# Windows

| 应用程序    | 描述                                     |
| :---------- | :--------------------------------------- |
| NETSTAT.EXE | 显示协议统计信息和当前 TCP/IP 网络连接。 |
| sc.exe      | 服务设置                                 |
| chcp.com    | 显示或设置活动代码页编号                 |
| findstr.exe | 在文件中寻找字符串                       |

```powershell
# 查看网络端口
NETSTAT.EXE -ano
```
